Desmos Supports for Coding in p5.js

Using Desmos Activity Builder to help students create and place shapes using p5.js

As a part of CS4All NYC, I teach 6th graders Computer Science/Coding using p5.js with the Creative Web curriculum.

Here are some examples of my students' work:

  • Robot or Car Project: Fall 2021 / Spring 2022
  • Emoji Project: Fall 2021 / Spring 2022

To support my students with expressing themselves with p5, I have written Desmos activities to help them understand the p5 coordinate grid and visualize the basic shapes and place them in their p5 sketches.

Please feel free to use and/or modify any of the following materials. If you'd like to contact me, please send me a message using the form below. Thanks!
